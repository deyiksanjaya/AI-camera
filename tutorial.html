System Setup

API Configuration

Lucid relies on Google's Gemini API. Follow these steps to connect your app:

  1. Visit aistudio.google.com and sign in.
  2. Go to "Get API Key" and create a new key.
  3. Open Lucid's settings, and paste the key.
  4. Set the model to gemini-3.1-flash-image-preview.

Prompt Management

Lucid utilizes a dual-memory system. Prompts for Real ➔ AI and AI ➔ Real are saved entirely separate from each other to prevent accidental cross-contamination.

  • Color Coding: Yellow UI indicates you are editing a Real ➔ AI prompt. Blue UI indicates an AI ➔ Real prompt.
  • Saving Drafts: Click "Save as Draft" to store frequent prompts. The list below will filter based on the active mode.
  • Invisible Directives: Lucid automatically appends hidden instructions (e.g., "photo realistic, keep other details exactly the same") to ensure the AI output looks like an unedited photograph.

Transition Dynamics

  • Trigger: Choose Tap for a standard touch, or Swipe Up to mimic a magical gesture/wave over the device.
  • Effect: Choose Fade for a slow, morphing change (ideal for mentalism), or Instant for a snappy, jarring change (ideal for visual magic).
  • Delay: Set a timer (in seconds) after the trigger is activated. This allows you to hand the phone to a spectator *after* triggering, letting the magic happen in their hands.

Technicals & Security
Covert Status Indicators

Knowing when the AI has finished processing without alerting the spectator is crucial. Lucid employs two virtually invisible indicators:

  • A
    The Flash 'A' Indicator During the main camera view, a tiny 'A' (Auto) text next to the flash icon appears. Once generation is complete, the 'A' quietly vanishes.
  • The Micro-Dot If you open the gallery prematurely, a microscopic, semi-transparent dot appears in the lower-right area of the screen. When it disappears, it's safe to transition.
Memory Clean-Up & Stealth

Auto Wipe (Memory Clean-up): Upon completion of the visual transition, the pre-transition image (the "evidence") is permanently purged from local storage and RAM. If a spectator handles the device post-effect, they will only find the final, clean image. Closing and reopening the app will not resurrect the original image.

State Persistence: Pre-captured images in AI ➔ Real mode are cached securely. You can lock your phone or casually switch apps; the setup remains intact and ready in the background until the trick is formally executed.

Blackout Mode: When enabled, the screen automatically fades to pitch black after a configured delay post-transition. This forces the device into a "sleep-like" state, allowing you to casually pocket the device without exposing the web browser interface or risking accidental touches.
